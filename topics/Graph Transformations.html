Graph Transformations


If we have a function $y = f(x)$ then how will we plot $y = f(x+a)$, $y = af(x)$, $y = f(ax)$ and $y = af(bx+c) +d $
  1. If $y = f(x)$ is given, plot $y = f(x) \pm c$ (Vertical shift)


  2. If $y = f(x)$ is given, plot $y = f(x\pm a)$ (Horizontal shift)


  3. If $y = f(x)$ is given, plot $y = a*f(x)$ (Vertical Stretch)
