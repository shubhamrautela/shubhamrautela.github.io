Functions


  • Definition of a function:- A relation from $A$ to $B$ ie. a subset of $A\times B$ is called a function from $A$ to $B$ if:
    1. for each $a\in A$ there exist $b \in B$ such that $(a,b) \in f$
    2. if $(a,b)\in f$ and $(a,c) \in f$ then $b=c$
  • if $(a,b) \in f$ then b is called image of $a$ under $f$
  • $f:A\rightarrow B$
    • A is domain of $f$
    • B is co-domain of $f$
    • $f(A)$ = range of $f$
    • $f(A) \subseteq B$
  • Equal Functions:- two functions $f$ and $g$ are equal iff
    1. domain of $f$ = domain of $g$
    2. co-domain of $f$ = codomain of $g$
    3. $f(x) = g(x)$ for every $x$ in their common domain.
    • $f: R - \{2\} \rightarrow R$, $f(x) = \frac{x^2-4}{x-2}$ and $g: R\rightarrow R$, $g(x) = x +2$
      $f(x) = g(x)$ but domain of $f(x) \neq g(x)$ so $f(x) \& g(x)$ are not equal functions.
    • $f=\{(n,n^2):n\in Z\}$, $g=\{(n,\mid{n^2}\mid):n\in Z\}$, $f=g$
  • Real functions: A function $f: A\rightarrow B$ is called a real valued function if B is a subset of $R$. If $A$ and $B$ both are subsets of R then it is a real function.
  • Domain of real functions: domain of f(x) is a set of all those real numbers for which $f(x)$ is meaningful.
    • if $f(x) = \frac{1}{x}$ then $x \neq 0$
    • if $f(x) = \sqrt{x-a}$ then $x-a\geq 0$
    • if $f(x) = \frac{1}{\sqrt{x-a}}$ then $x-a\gt 0$
    • if $f(x)= \sqrt{4-x} + \frac{1}{\sqrt{x^2-1}}$
      $\Rightarrow (4-x)\geq 0$ and $(x^2-1) \gt 0$
      $\Rightarrow x = (-\infty,-1) \cup (1,4]$
  • Range of a function: Range of a real function is the set of all real values taken by f(x) at points in its domain.
    put $y = f(x)$
    $ \Rightarrow$ solve for $x$ in terms of $y$
    $ \Rightarrow$ find values of $y$ for which the values of $x$ are real in its domain.
    • $f(x) = \frac{1}{\sqrt{x-5}}$
      domain = $x > 5$, $x\in (5,\infty)$
      $x-5>0 \Rightarrow \sqrt{x-5} > 0 \Rightarrow \frac{1}{\sqrt{x-5}} \Rightarrow f(x) > 0$
      Range of $f = (0,\infty)$
    • $f(x) = \frac{3}{2-x^2}$
      Domain($f$) = $R-\{-\sqrt{2},\sqrt{2}\}$
      $y = \frac{3}{2-x^2}$
      $\Rightarrow 2y - x^2y = 3$
      $\Rightarrow x = \pm \sqrt{\frac{2y-3}{y}}$
      $\Rightarrow \frac{2y-3}{y} \geq 0$
      $\Rightarrow y \in (-\infty,0) \cup [\frac{3}{2},\infty)$
    • $f(x) = \frac{x}{1+x^2}$
      Domain($f$) = $R$
      $y = \frac{x}{1+x^2}$
      $\Rightarrow x^2 y - x + y = 0$
      $\Rightarrow x = \frac{1\pm \sqrt{1-4y^2}}{2y}$
      $x$ will assume real values, if $1 - 4y^2 \geq 0$ and $y\neq 0$
      $\Rightarrow y \in [-\frac{1}{2},\frac{1}{2}] - \{0\}$
      Also, $y=0$ for x =0, Range($f$) = $[-\frac{1}{2},\frac{1}{2}]$
    • $f(x) = \frac{1}{2-sin3x}$
      $2-sin3x \neq 0$ for any $x\in R$
      $f(x) = \frac{1}{2-sin3x}$ is defined for all $x \in R$
      domain($f$) = $R$
      Range of $f$:
      $1 \leq 2-sin3x \leq 3$ for all $x\in R$
      $\Rightarrow \frac{1}{3} \leq \frac{1}{2-sin3x} \leq 1$ for all $x \in R$
      $\Rightarrow f(x) \in [\frac{1}{3},1]$
  • Constant function:- $f(x) = k, x \in R$
    domain of $f = R$, Range of $f(x) = {k}$
  • Identity function:- the function that associate each real number to itself denoted by I.
    $f(x) = x$, $I:R\rightarrow R$, $I(x) = x$, $x\in R$
  • Modulus function:- $f(x) = |x|$
    domain of $f = R$, range of $f = R^{+} \{x:x\geq 0\}$
    Properties of modulus function:-
    1. $\sqrt{x^2} = |x| \Rightarrow y^2 = x \Rightarrow y = \pm \sqrt{x}$
    2. if $a$ and $b$ are positive real numbers
      $x^2 \leq a^2 \Leftrightarrow |x| \leq a \Leftrightarrow -a \leq x \leq a$
      $x^2 \geq a^2 \Leftrightarrow |x| \geq a \Leftrightarrow -a \geq x \geq a$
    3. $|x+y| = |x| + |y|$ only if $x$ & $y \geq 0$ or $x$ & $y < 0$
    4. $|x-y| = |x| - |y|$ only if $x$ & $y \geq 0$ & $|x| \geq |y|$ or $x$ & $y < 0, |x| \geq |y|$
    5. $|x\pm y| \leq |x| + |y|$
    6. $|x\pm y| \geq ||x|-|y||$

  • Greatest Integer function(floor function)
    $\lfloor x \rfloor$ greatest integer less than equal to x.
    $f: R \rightarrow R, f(x) = \lfloor x \rfloor \forall x \in R$
    domain:- $R$, range:- $Z$
    if $n$ is integer and $x$ lies between $n$ & $n+1$
    1. $\lfloor n \rfloor = -\lfloor n \rfloor$
    2. $\lfloor x + k \rfloor = \lfloor x \rfloor + k$
    3. $\lfloor -x \rfloor = -\lfloor x \rfloor - 1$
    4. $\lfloor x \rfloor + \lfloor -x \rfloor$ = -1 if $x \in Z $ = 0 if $x \in z$
    5. $\lfloor x \rfloor - \lfloor -x \rfloor$ = $2x+1$ if $x \in Z $ = 2x if $x \in z$
    6. $\lfloor x \rfloor \geq k \Rightarrow x \geq k$
    7. $\lfloor x \rfloor \leq k \Rightarrow x \leq k+1$
    8. $\lfloor x \rfloor \gt k \Rightarrow x \geq k+1$
    9. $\lfloor x \rfloor \lt k \Rightarrow x \lt k$
    10. $\lfloor x+y \rfloor = \lfloor x \rfloor + \lfloor y+x - \lfloor x \rfloor \rfloor$
    11. $\lfloor x \rfloor \lfloor x+\frac{1}{n} \rfloor + \lfloor x+\frac{2}{n} \rfloor + ... + \lfloor x+\frac{n-1}{n} \rfloor$ = $\lfloor nx \rfloor$
  • Ceiling Funciton:- $f:R\rightarrow R$ $f(x) = \lceil x \rceil$
    domain: $R$, Range: $Z$
    Properties:-
    1. $\lceil -n \rceil = -\lceil n \rceil$
    2. $\lceil -x \rceil = -\lceil x \rceil + 1$
    3. $\lceil x + n \rceil = \lceil -x \rceil + n$ where $x \in R - Z$ and $n \in Z$
    4. $\lceil x \rceil + \lceil -x \rceil = 1, x \not\in Z, = 0, x \in Z$
    5. $\lceil x \rceil - \lceil -x \rceil = 2 \lceil x \rceil -1, x \not\in Z, = 2 \lceil x \rceil $
  • Signum function:- $f(x) = \frac{|x|}{x} x\neq 0, 0 x=0$
    $f(x)=\{1(x\gt 0),0(x=0),-1(x\lt 0\}$
  • Exponential function:- if $a$ is a positive real number $\neq 1$ then function $x\rightarrow a^x$ is called the exponential function.
    $f: R \rightarrow R$, $f(x) = a^x, a>0 a\neq 1$
    domain of exp. function = $R$, range is set $(0,\infty)$
    1. Case I: when $a>1, a^x$ increases as x increases
      $f(x) = \left\{\begin{array}{ll} \lt 1 & : x \lt 0\\ =1 & : x = 0 \\ \gt 1 & : x > 1 \end{array} \right.$
    2. Case II: when $a\lt 1, a \gt 0, a^x$
      $f(x) = \left\{\begin{array}{ll} \lt 1 & : x \gt 0\\ =1 & : x = 0 \\ \gt 1 & : x \lt 1 \end{array} \right.$
  • Logarithmic function:- $a>0, a\not\in 1 f(x) = log_ax, x>0$
    $log_ax = y \Rightarrow x = a^y$
    1. Case I: when $a>1$
      $f(x) = log_ax \left\{\begin{array}{ll} \lt 0 & : 0 \lt x \lt 1\\ =0 & : x = 1 \\ \gt 0 & : x > 1 \end{array} \right.$
    2. Case II: when $a\lt 1, a \gt 0, a^x$
      $f(x) = log_ax \left\{\begin{array}{ll} \gt 0 & : 0 \lt x \lt 1\\ =0 & : x = 1 \\ \lt 0 & : x \gt 1 \end{array} \right.$
    Properties:-
    1. $log_a1 = 0, a\gt 0, a\neq 1$
    2. $log_aa = 1$
    3. $log_a(xy) = log_a|x| + log_a |y|$
    4. $log_a(\frac{x}{y}) = log_a|x| - log_a |y|$
    5. $log_ax^n = nlog_ax$
    6. $log_{a^n} x^m = \frac{m}{n}log_ax$
    7. $x^{log_ay-}= y^{log_ax}$
    8. $log_ax = \frac{1}{log_xa}$
