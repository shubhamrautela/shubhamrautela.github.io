Sets


  • Set:- A well defined & well distinguished collection of objects/elements is called a set.
  • Known sets notation
    • N:- Natural Numbers
    • W:- Whole Numbers
    • Z:- Integers
    • R:- Real Numbers
    • Q:- Rational Numbers
    • C:- Complex Numbers
  • x belongs to A, $x \epsilon A$, $2 \epsilon A$ if $A = \{1, 2, 3, 4\}$
  • Representation of Sets
    • Roster form: To explicitly write out the elements. Eg:- {1, 2, 3, 4, 5}
    • Set Builder form: To write the set in a compact algebraic form. Eg:- {x:(such that) x < 5, x $\epsilon$ N}
  • Order doesn't matter. Eg:- {4, 1, 3, 2} = {1, 2, 3, 4}
  • Repitition doesn't matter. Eg:- {F, O, L, L, O, W} = {W, O, L, F}
  • Types of sets
    • Null set:- A set having no elements. Also called an Empty set. { }, $\phi$
    • Cardinal number:- The number of elements in a set is called its cardinality. If $A = {1, 2, 3}$ then $n(A) = 3$
    • Singleton set:- A set having only one element. Eg:- A set containing even prime numbers.
    • Finite & Infinite sets:-
      1. {$x: x \epsilon N, x < 100 $}
      2. {$x: x \epsilon N, x > 2 $}
    • Universal set:- A set containing all possible elements in the given context. Represented by $U$.
