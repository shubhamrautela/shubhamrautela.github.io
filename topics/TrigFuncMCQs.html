  4. If $sinx = cos^2x$, then write the value of $cos^2x(1+cos^2x)$.

    $cos^2x(1+cos^2x)$

  5. If $sinx + cosecx = 2$, then write the value of $sin^nx+cosec^nx$.

  6. If $sinx + sin^2x = 1$, then write the value of $cos^{12}x + 3cos^{10}x + 3cos^8x + cos^6x$.

    $cos^{12}x + 3cos^{10}x + 3cos^8x + cos^6x = cos^6x(cos^6x + 3cos^4x + 3cos^2x + 1)$
    $\Rightarrow cos^6x((cos^2x + 1)^3)$
    $\Rightarrow (cos^2x)^3((sinx + 1)^3)$
    $\Rightarrow ((sinx)(sinx + 1))^3$
    $\Rightarrow (sin^2x + sinx)^3$
    $\Rightarrow (1)^3$
    $\Rightarrow 1$

  7. If $sinx + sin^2x = 1$, then write the value of $cos^{8}x + 2cos^{6}x + cos^4x$.

    $cos^{8}x + 2cos^{6}x + cos^4x = cos^4x(cos^4x + 2cos^2x + 1)$
    $\Rightarrow cos^4x((cos^2x + 1)^2)$
    $\Rightarrow (cos^2x)^2((sinx + 1)^2)$
    $\Rightarrow ((sinx)(sinx + 1))^2$
    $\Rightarrow (sin^2x + sinx)^2$
    $\Rightarrow (1)^2$
    $\Rightarrow 1$

  8. If $sin\theta_1 + sin\theta_2 + sin\theta_3 = 1$, then write the value of $cos\theta_1 + cos\theta_2 + cos\theta_3$.
