Sets


  • Set:- A well defined & well distinguished collection of objects/elements is called a set.
  • Known sets notation
    • N:- Natural Numbers
    • W:- Whole Numbers
    • Z:- Integers
    • R:- Real Numbers
    • Q:- Rational Numbers
    • C:- Complex Numbers
  • x belongs to A, $x \epsilon A$, $2 \epsilon A$ if $A = \{1, 2, 3, 4\}$
  • Representation of Sets
    • Roster form: To explicitly write out the elements. Eg:- {1, 2, 3, 4, 5}
    • Set Builder form: To write the set in a compact algebraic form. Eg:- {x:(such that) x < 5, x $\epsilon$ N}
  • Order doesn't matter. Eg:- {4, 1, 3, 2} = {1, 2, 3, 4}
  • Repitition doesn't matter. Eg:- {F, O, L, L, O, W} = {W, O, L, F}
  • Cardinal number:- The number of elements in a set is called its cardinality. If $A = {1, 2, 3}$ then $n(A) = 3$
  • Types of sets
    • Null set:- A set having no elements. Also called an Empty set. { }, $\phi$
    • Singleton set:- A set having only one element. Eg:- A set containing even prime numbers.
    • Finite & Infinite sets:-
      1. {$x: x \epsilon N, x < 100 $}
      2. {$x: x \epsilon N, x > 2 $}
    • Universal set:- A set containing all possible elements in the given context. Represented by $U$.
    • Equal sets:- Two sets are called equal if they have exactly same elements. Eg:- $A = \{4, 1, 3, 2\}, B = \{1, 2, 3, 4\}, A = B$
    • Equivalent sets:- Two sets are called equivalent if they have exactly the same number of elements. Eg:- $A = \{M, A, R, X\}, B = \{1, 2, 3, 4\}, n(A) = n(B)$
  • Subset:- $A$ is a subset of $B$ if all the elements of $A$ are in $B$. $A \subseteq B$
  • Proper Subset:- $A$ is a proper subset of $B$ if $B$ has some extra elements that aren't $A$. $A \subset B$
  • Some nots on subsets
    1. Null set is a subset of every set.
    2. Every set is a subset of Universal set.
    3. Every set is a subset of itself. $A \subset A$
  • Number of subsets of a set having n elements is $2^n$. Proof: There are $\sum^{n}_{r=0} {^nC_r}$ ways of choosing the elements out of $n$ elements, which is equal to ${^nC_0}+{^nC_1}+...+{^nC_n} = (1 + 1)^n = 2^n$
  • Power Set:- A set containing all possible subsets of a set. If $A = \{1, 2, 3, 4, 5\}, n(P(A)) = 2^3$
  • Operation Definition Venn Diagram Properties
    Union $\cup$ All the elements of the sets.
    • $A \cup B = B \cup A$
    • $A \cup A = A$
    • $A \cup (B\cup C) = (A\cup B)\cup C$
    • $A \cup U = U$
    • $A \cup \phi = A$
    • if $A \subseteq B$ then $A \cup B = B$
    Intersection $\cap$ All the elements commmon to the sets.
    • $A \cap B = B \cap A$
    • $A \cap A = A$
    • $A \cap (B\cap C) = (A\cap B)\cap C$
    • $A \cap U = A$
    • $A \cap \phi = \phi$
    • if $A \subseteq B$ then $A \cap B = A$
    Difference Elements in a set minus the elements in another set
    • $A - A = \phi$
    • $A - U = \phi$
    • $A - \phi = A$
    • $A - (B\cap C) = (A - B) \cup (A - C)$
    • $A - (B\cup C) = (A - B) \cap (A - C)$
    Complement $^{c}$ Everything except the elements in the set.
    • $A \cup A^{c} = U$
    • $A \cap A^{c} = \phi$
    • $U^{c} = \phi$
    • $\phi^{c}=U$
    • $A - A^{c} = A$
    Symmetric Difference $\triangle$ Whole minus the common elements.
    • $A \triangle B = (A \cup B) - (A \cap B)$
  • $n(A\cup B)= n(A)+ n(B)- n(A\cap B)$
  • $n(A\cup B \cup C)= n(A)+ n(B) + n(C)- n(A \cap B) - n(B \cap C) - n(A \cap C) + n(A \cap B \cap C)$
